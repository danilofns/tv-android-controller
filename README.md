# TV Android Controller

Projeto para controle e interação com dispositivos Android TV de forma simples e programável.

## 📌 Descrição

O **tv-android-controller** tem como objetivo fornecer uma interface para conectar, controlar e automatizar ações em dispositivos Android TV, permitindo executar comandos remotos, navegar pela interface e integrar funcionalidades com outras aplicações ou sistemas.

## ✨ Funcionalidades iniciais

* Conexão com dispositivos Android TV [Em desenvolvimento]
* Envio de comandos remotos [Em desenvolvimento]
* Navegação entre telas [Em desenvolvimento] 
* Simulação de ações do controle (OK, voltar, home, volume, etc.) [Em desenvolvimento]
* Estrutura preparada para automações futuras [Em desenvolvimento]
